# 原始〜平安 記事シリーズ

## 目的

YouTubeベース詳細稿 `01-japan-creation-and-dawn-of-transformation` を、SEOと読みやすさを考えて親記事＋個別記事に分割した作業フォルダ。

## 記事構成

1. `01-japan-creation-and-dawn-of-transformation`
   - 親記事: 日本創生と変革の夜明け
   - 役割: 原始〜平安の全体像を見せ、各個別記事へ内部リンクする入口

2. `02-jomon-to-yayoi-rice-changed-japan`
   - 縄文から弥生へ: 米づくりが日本を変えた
   - 役割: 稲作、定住、貧富差、小国乱立への流れ

3. `03-yamataikoku-and-himiko`
   - 邪馬台国と卑弥呼: 日本の始まりに残る最大の謎
   - 役割: 魏志倭人伝、卑弥呼、九州説・近畿説

4. `04-kofun-and-yamato-state`
   - 古墳とヤマト政権: 巨大な墓が語る国家の誕生
   - 役割: 古墳、豪族連合、ヤマト政権

5. `05-buddhism-soga-and-statecraft`
   - 仏教伝来と蘇我氏: 日本はなぜ大陸の制度を受け入れたのか
   - 役割: 仏教、蘇我氏、物部氏、推古天皇、聖徳太子

6. `06-taika-reform-and-ritsuryo-state`
   - 大化の改新と律令国家: 日本が「国」になろうとした日
   - 役割: 645年、大化、壬申の乱、天武天皇、古事記・日本書紀

7. `07-nara-heian-buddhism-and-capital`
   - 奈良から平安へ: 仏教国家の光と影
   - 役割: 平城京、東大寺、道鏡、平安京、最澄・空海、阿弖流為

8. `08-heian-samurai-rise`
   - 平安の裏側: 荘園、武士、そして鎌倉への道
   - 役割: 荘園、平将門、藤原道長、院政、平清盛

## 作業ルール

- `drafts/`: 日本語ドラフトとWord版を置く。
- `references/`: 購入書籍、論文、PDF、URLメモを置く。
- `assets/`: 父に入れてもらう写真候補を置く。
- `Eiji_check/`: 史実確認を依頼する完成寄りの日本語版だけを置く。
- `translate/`: チェック後の英語・ドイツ語翻訳用ファイルを置く。

1. 各記事の史実確認ポイントを資料で確認する。
2. 記事ごとに写真指定を具体化する。
3. 親記事から個別記事へ、個別記事同士も自然に内部リンクする。
4. 確認済み原稿だけを `Eiji_check/` に入れる。
