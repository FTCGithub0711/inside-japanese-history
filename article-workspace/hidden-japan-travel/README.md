# Hidden Japan Travel Workspace

このフォルダは、`Hidden Japan Travel` 系の記事制作作業場です。

## 構成

- `01-hidden-historic-places-in-japan/` — 親記事。混雑回避・穴場史跡・歴史起点旅行の入口。
- `02-odawara-castle-history-travel/` — 個別記事。小田原城の詳しい歴史、観光方法、食事、ホテル導線。

各記事フォルダでは、以下を使います。

- `references/` — 公式観光情報、自治体資料、博物館ページ、論文、書籍メモ
- `notes/` — 調査メモ、候補地比較、SEOメモ
- `extracted/` — 資料から抜き出した論点
- `drafts/` — 英語・ドイツ語の草稿
- `assets/` — 写真、地図、画像メモ

## 編集ルール

- 観光情報だけで終わらせない。
- 先に歴史的意味を説明し、その後に旅行導線を置く。
- 有名観光地より、混雑を避けたい読者が選ぶ理由を重視する。
- 各候補地には、公式情報・地図・宿泊検索・内部リンク候補をセットで整理する。

## 親記事と個別記事の座組

親記事は入口記事として使う。例: `Hidden Historic Places in Japan`

親記事内で `Odawara Castle`、`Sekigahara Battlefield`、`Hachigata Castle Ruins` などの名前が出たら、将来的にそれぞれの個別記事へリンクする。

個別記事では以下を必ず扱う。

1. 詳しい歴史
2. 現地で見るべきポイント
3. 具体的な観光の仕方
4. アクセスと所要時間
5. 美味しい食事どころ
6. おすすめホテル・宿泊エリア
7. 写真を入れる場所
8. 関連する親記事・他スポット記事への内部リンク

食事どころとホテルは情報が変わりやすいため、記事作成時に最新情報を確認してから入れる。父チェックでは「歴史・写真・現地感」を優先して確認する。
