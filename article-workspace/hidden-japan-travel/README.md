# Hidden Japan Travel Workspace

このフォルダは、`Hidden Japan Travel` 系の記事制作作業場です。

## 構成

- `01-hidden-historic-places-in-japan/` — 親記事。混雑回避・穴場史跡・歴史起点旅行の入口。

各記事フォルダでは、以下を使います。

- `references/` — 公式観光情報、自治体資料、博物館ページ、論文、書籍メモ
- `notes/` — 調査メモ、候補地比較、SEOメモ
- `extracted/` — 資料から抜き出した論点
- `drafts/` — 英語・ドイツ語の草稿
- `assets/` — 写真、地図、画像メモ

## 編集ルール

- 観光情報だけで終わらせない。
- 先に歴史的意味を説明し、その後に旅行導線を置く。
- 有名観光地より、混雑を避けたい読者が選ぶ理由を重視する。
- 各候補地には、公式情報・地図・宿泊検索・内部リンク候補をセットで整理する。
