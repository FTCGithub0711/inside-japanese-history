# Eiji Check Folder

このフォルダは、Eijiさんに内容確認を依頼するための日本語資料を置く場所です。

## 使い方

1. Codexが日本語アウトラインまたは日本語原稿を作成する。
2. `.docx` をこのフォルダに入れる。
3. Futoshiさんがこの `.docx` をEijiさんへ共有する。
4. Eijiさんが内容、違和感、写真候補、歴史的な補足を確認する。
5. チェック済みファイルは `../translate/` に移動またはアップロードする。
