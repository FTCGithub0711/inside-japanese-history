# Drafts

このフォルダは、日本語ドラフトを書くための作業場所です。

## 使い方

1. `hidden_historic_places_draft_ja.md` を上から埋めてください。
2. 最初は文章として完成していなくても問題ありません。箇条書き、メモ、写真指定だけでも大丈夫です。
3. 史実・地名・人名・年代に不安がある箇所は、本文中に `[要確認: ...]` と書いてください。
4. 写真を入れたい場所には `[写真候補: ...]` と書いてください。
5. 一通り書けたら、こちらで構成、SEO、日本語の自然さ、翻訳しやすさを確認します。

## Eijiチェックに出すタイミング

Eijiチェックは、本文ドラフトが一通り完成し、写真指定も入った後に行います。
この段階では、アウトラインだけではなく「読める記事」に近い状態まで整えてから渡します。
