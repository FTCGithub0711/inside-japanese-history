# Translate Folder

このフォルダは、Eijiさん確認後の日本語原稿を置く場所です。

## 使い方

1. `../Eiji_check/` のdocxをEijiさんに確認してもらう。
2. 修正済み・確認済みのdocxまたはMarkdownをこのフォルダに入れる。
3. Codexはこのフォルダの最新版をもとに、英語記事を作成する。
4. 英語記事化の際は、本文、写真位置、参考文献、旅行導線、アフィリエイト導線を整理する。

## 命名例

- `hidden_historic_places_checked_ja.docx`
- `hidden_historic_places_checked_ja.md`
- `hidden_historic_places_photo_notes.md`
