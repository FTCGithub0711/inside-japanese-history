# Article Workspace

このフォルダは、公開前の記事制作作業場です。購入済みの電子書籍、論文PDF、メモ、画像、草稿を記事ごとに分けて置きます。

## フォルダ構成

- `top-topics/` — まず整える人気トピック記事10本

## 基本ルール

- 公開用のサイト実装は `site/` に置く。
- 参考資料、抜き書き、構成案、草稿は `article-workspace/` に置く。
- 購入した電子書籍や論文は、該当記事の `references/` に入れる。
- 画像は `assets/` に入れる。
- 私が読むための補足メモは `notes/` に入れる。

## 注意

購入済み資料は記事作成のために要約・論点抽出する。本文に長い引用は入れず、必要な場合だけ短く引用し、基本は自分たちの言葉で再構成する。
