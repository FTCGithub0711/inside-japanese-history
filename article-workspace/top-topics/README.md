# Top Topics Article Workspace

英語圏で検索需要がある人気トピック10本の記事作業場です。

## 使い方

1. 該当記事フォルダを開く。
2. 原則として2020年以降のPDF、EPUB、論文、スクリーンショットを `references/` に入れる。
3. どの資料を入れたか、各記事フォルダの `source_index.md` に書く。
4. 自分のメモや父のメモは `notes/` に入れる。
5. 私に「このフォルダを読んで構成を作って」と指示する。
6. 私が `extracted/` に論点抽出、`drafts/` に構成案・草稿を作る。

## 参考文献のルール

- 購入・アップロード優先は2020年以降。
- 古い文献は、現在でも基礎文献として使う理由がある場合だけ `source_index.md` の Notes に `例外基礎文献` と書く。
- 古い文献だけで記事を作らない。2020年以降の資料が不足する場合は、先に追加調査する。
- 具体的な候補は `seo-research/09_japanese_reference_sources_for_top_topics.md` を正本にする。

## サブフォルダ

- `references/` — 購入済み電子書籍、論文、PDF、EPUB
- `notes/` — 自分のメモ、父のメモ、疑問点
- `extracted/` — 私が作る抜き書き・論点整理
- `drafts/` — 構成案、英語草稿、日本語構成メモ
- `assets/` — 記事用画像、地図、写真

## ファイル名ルール

推奨:

```text
author-year-short-title.pdf
author-year-short-title.epub
father-notes-YYYY-MM-DD.md
my-questions-YYYY-MM-DD.md
```

例:

```text
kasaya-2022-sekigahara.pdf
yoshimaru-2022-ninja.pdf
father-notes-2026-05-05.md
```
