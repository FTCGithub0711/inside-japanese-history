# Task Folder

このフォルダは、実際に動くための作業スケジュールとチェックリストを置く場所です。

- `article_creation_schedule_ja.md` — 記事作成・公開ペース・SEO方針・父の記事の扱い
- `affiliate_setup_checklist_ja.md` — 旅行・書籍アフィリエイト導入チェックリスト
- `site_launch_checklist_ja.md` — ドメイン、デプロイ、Search Console、Analyticsの公開準備

戦略や調査資料は `seo-research/`、実行順序や日々の作業は `task/` に置きます。
