Put hero slideshow photos in this folder.

Recommended:
- JPG or WebP
- 1800px wide or larger
- Landscape orientation
- Descriptive lowercase filenames, e.g. `hachigata-castle-01.jpg`

After adding a file, register it in `site/src/consts.ts`:

```ts
{ src: '/images/hero/hachigata-castle-01.jpg', credit: 'Photo: Eiji Iwata' }
```
